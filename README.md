# UGR-Net: Uncertainty-Gated Multi-Scale Fusion with Randomized Data-Feature Augmentation for Computer-AidedDiagnosis of Glaucoma This is the official pytorch implementation of our Media paper "UGR-Net: Uncertainty-Gated Multi-Scale Fusion with Randomized Data-Feature Augmentation for Computer-AidedDiagnosis of Glaucoma". In this paper, The framework comprises four components: (1) Data– Feature Augmentation with Domain Randomization (DFA), which injects per-sample and per-iteration randomization at both the image and shallow-feature levels to broaden the style space; (2) an Explicit–Implicit Augmentation Randomization (EI) module that combines explicit channel-moment mixing with implicit moment transfer to expand style diversity in a controlled manner; (3) Deep– Shallow Feature Fusion (DS), which aligns and fuses shallow details with deep semantics to build multi-scale representations; and (4) an Uncertainty-Gated Attention (UGA) module that derives evidential uncertainty from Dirichlet strength and applies reliability-aware soft gating across channels and spatial maps, suppressing spurious responses while preserving subtle glaucomatous cues. # Requirements python 3.8 pytorch 2.8.0 cuda 129 # Data Preparation https://zenodo.org/records/12562673
